Git Commands Reference Guide

Name Command Description Action
Push git push <remote> <branch> Upload local changes to the remote repository.
Push All git push --all <remote> Push all branches to the remote.
Pull git pull <remote> <branch> Fetch and integrate changes from the remote.
Clone git clone <repo-url> Create a local copy of a remote repository.
Commit git commit -m "message" Record changes to the repository.
Add git add <file> Add changes to the staging area.
Status git status Show the working tree status.
Branch git branch <branch-name> Create a new branch.
Merge git merge <branch> Join branches together.
Checkout git checkout <branch-name> Switch branches or restore files.
Log git log Show commit logs.
Init git init Initialize a new Git repository.
Fetch git fetch <remote> Download objects and refs from another repository.
Rebase git rebase <branch> Reapply commits on top of another base tip.
Stash git stash Temporarily store changes.
